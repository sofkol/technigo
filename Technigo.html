What is "Responsive design"?

You might have heard of "responsive design" before, but what does it actually mean? I'll give you a simple explanation and some useful links.

Responsive web design is about using HTML and CSS to automatically resize, hide, shrink, or enlarge, a website, to make it look good on all devices (desktops, tablets, and phones). Responsive web design make web pages look good on all devices. Responsive web design has become more important as the amount of mobile device user has come to account for more than half of total internet traffic
